HOW TO ADD PHOTOS
==================

Team photos go in: images/team/
  kanayeva.jpg       -> Dr. Damira Kanayeva
  sagdat.jpg         -> Kamila Sagdat
  issimova.jpg       -> Meruyet Issimova
  zhumabekova.jpg    -> Zhazira Zhumabekova
  kozybayeva.jpg     -> Kamshat Kozybayeva

  Tip: square photos work best (e.g. 500x500px), cropped to the face/shoulders.

Lab photos go in: images/lab/
  bench.jpg
  potentiostat.jpg
  team-working.jpg
  equipment.jpg

  Tip: landscape photos work best (4:3 ratio, e.g. 1200x900px).

That's it — just add a file with the exact name above into the matching folder,
and it will automatically appear on the website. No code editing needed.
If a photo is missing, the page shows a clean placeholder instead of a broken image.
